# LemonadeChains

LemonadeChains is an educational market trading simulator specifically designed to educate users on the intricacies behind stock option contracts, including call/put contracts and a strategy building page that allows you to visualize the profit/loss of combining multiple option contracts at once.

## Features

- **Market Trading With Real-Time Information** – Pull stock ticker and list of option contracts directly from Yahoo Finance using the yFinance library, allowing users to trade a virtual portfolio on this dynamically updating data  
- **Educational Articles** – Through LemonadeLearn program, users can read through a sequence of focused articles on all the important information behind trading stock option contracts efficiently  
- **Smart Option Contract Pricing** – Generated prices that relate time with intrinsic value for stock options contracts using the Black-Scholes model for European-style options  

## Tech Stack

- **Frontend:** HTML/CSS (Made using Bootstrap)  
- **Backend:** Python (Flask)  
- **Stock Ticker/Options Data:** yFinance Python Library  
- **Database:** Google Firebase
